## 3. 论文方法梳理

### 3.1 问题背景

普通对抗样本会在图像中加入微小扰动，让分类模型给出错误预测；但扰动通常不可逆，图像原始内容不能无损恢复。可逆对抗攻击希望同时满足两件事：

1. 攻击阶段：生成能误导分类模型的对抗图像。
2. 恢复阶段：合法接收方可以从对抗图像中恢复原始图像。

这篇论文的重点不是单纯提高攻击成功率，而是在 YUV 颜色空间中减少无效扰动，并增强跨模型迁移性。

### 3.2 核心思想

论文利用 YUV 颜色空间把图像分成亮度通道 Y 和色度通道 U/V：

1. **在 Y 通道生成对抗扰动**  
   人眼和分类模型对亮度信息更敏感，论文使用 YFGSM、YI-FGSM、YPGD、YMI-FGSM 等 Y 通道攻击策略，避免在 U/V 通道生成多余扰动。

2. **在 U/V 通道嵌入可逆恢复信息**  
   将恢复所需信息嵌入 U/V 通道，避免覆盖已经在 Y 通道生成的有效扰动。

3. **使用集成模型提升迁移性**  
   代码中的 `EnModel.py` 将 DenseNet161、Inception v3、GoogLeNet 的输出做平均，用于生成更不依赖单个白盒模型的扰动。

4. **使用 CAM 注意力区域**  
   主脚本通过 ResNet50 的 Grad-CAM++ 生成注意力掩码，只在关键区域替换 Y 通道扰动，降低图像失真。

### 3.3 代码流程

主入口：`reversible_attack_OURS.py`

流程大致如下：

1. 设置随机种子。
2. 自动选择 `cuda` 或 `cpu`。
3. 加载 Inception v3 作为分类验证模型。
4. 加载集成攻击模型 `EnModel`。
5. 遍历 `./ORI_IMG/` 中的图片。
6. 从文件名中解析 ImageNet 标签，例如 `0001_281.png` 中的 `281`。
7. 使用 CAM 生成注意力区域。
8. 使用 `atk_MI_YFGSM` 生成 Y 通道扰动。
9. 使用 `embed_main` 将恢复信息嵌入 U/V 通道。
10. 将生成的可逆对抗样本保存到 `./output/rae/`。

## 7. 4060 显存优化建议

如果 RTX 4060 显存不足或运行很慢，按以下顺序降负载：

1. `ORI_IMG` 中只保留 1 张图片先跑通。
2. 将 `atk_MI_YFGSM` 的 `steps=20` 暂时改为 `steps=5` 或 `steps=10`。
3. 将 `max_iteration in range(50)` 暂时改为 `range(5)`。
4. 如果集成模型显存压力太大，可临时修改 `EnModel.py`，只保留 Inception v3 或 GoogLeNet 做快速验证。
5. 使用任务管理器或 `nvidia-smi` 观察显存。

快速验证优先级高于复现完整论文表格。建议先得到 1 张 RAE 输出图，再逐步恢复原始参数。

## 8. 常见问题与处理

### 8.1 `git` 不是内部或外部命令

原因：本机未安装 Git，或 PATH 未配置。

处理：

1. 安装 Git for Windows。
2. 重新打开 PowerShell。
3. 或直接下载 ZIP。

### 8.2 `python` 打开 Microsoft Store

原因：Windows App Execution Alias 抢占了 `python` 命令。

处理：

1. 安装 Python 3.10/3.11。
2. 安装时勾选 `Add python.exe to PATH`。
3. 或在 Windows 设置中关闭 Python 的 App execution aliases。

### 8.3 `torch.cuda.is_available()` 为 `False`

可能原因：

1. 安装了 CPU 版 PyTorch。
2. NVIDIA 驱动过旧。
3. 虚拟环境中包版本混乱。

处理：

```powershell
pip uninstall -y torch torchvision torchaudio
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
```

如果仍失败，去 PyTorch 官网选择器重新生成命令。

### 8.4 输出目录不存在导致保存失败

处理：

```powershell
mkdir output
mkdir output\rae
```

### 8.5 文件名解析标签失败

主脚本需要文件名中有下划线和数字标签，例如：

```text
0001_281.png
```

如果命名为 `cat.png`，脚本会在 `filename.split('_')` 处失败。

### 8.6 图片尺寸相关错误

主脚本内部按 `299x299` 做多处处理。若输入图片尺寸不一致，可能触发索引错误或指标计算错误。建议先把图片统一为 `299x299`。

1. 使用 10-50 张 ImageNet 格式样例图。
2. 记录攻击成功率。
3. 记录 PSNR、SSIM、L2、Linf 等指标。
4. 比较不同步数 `steps=5/10/20` 的速度和成功率。

第三阶段扩展：

1. 单模型攻击 vs 集成模型攻击。

4. 对比只攻击 RGB、只攻击 Y 通道、Y 通道攻击 + UV 嵌入三种设置。
