## 安装问题与修复

### 问题 1：policy-refinement 安装时 README 编码错误

现象：

```text
UnicodeDecodeError: 'gbk' codec can't decode byte ...
```

原因：Windows 默认 GBK 编码读取 UTF-8 README。

修复：

```powershell
$env:PYTHONUTF8='1'
$env:PYTHONIOENCODING='utf-8'
.\.venv-policy\Scripts\python.exe -m pip install -e .\Rt-LRM-main\policy-refinement
```

### 问题 2：torch 2.12.0 在 Windows 上 DLL 初始化失败

现象：

```text
OSError: [WinError 1114] Error loading ... torch\lib\c10.dll
```

修复：卸载默认安装的 torch，改装 CPU 版。

```powershell
.\.venv-policy\Scripts\python.exe -m pip uninstall -y torch
.\.venv-policy\Scripts\python.exe -m pip install "torch==2.3.1+cpu" --index-url https://download.pytorch.org/whl/cpu --timeout 180 --retries 10 --no-cache-dir
```

### 问题 3：transformers 5.10.2 要求 PyTorch >= 2.4

现象：虽然环境测试可通过，但 transformers 会提示禁用 PyTorch。

修复：

```powershell
.\.venv-policy\Scripts\python.exe -m pip install "transformers==4.41.2" --force-reinstall
.\.venv-policy\Scripts\python.exe -m pip install "numpy==1.26.4" --force-reinstall
```
