#### 阶段 0 Paper 信息摘要

**What：** 论文提出基于 YUV 颜色空间的 error-free reversible adversarial attack，同时解决未授权模型识别干扰和授权方原图无损恢复的问题，并重点处理跨模型迁移性不足与无效扰动问题。

**How：** 方法将攻击扰动集中在 Y 亮度通道，将恢复信息嵌入 U/V 色度通道，并使用 DenseNet161、Inception v3、GoogLeNet 等模型组成集成攻击模型；主脚本还使用 ResNet50 + Grad-CAM++ 约束扰动区域。

**Why：** Y 通道更集中表达亮度、结构和纹理信息，适合生成高效攻击扰动；U/V 通道用于嵌入恢复信息，可避免覆盖 Y 通道扰动；集成模型降低对单一模型的过拟合，从而增强迁移性。

**Pros：** 机制可解释，通道分工明确；兼顾攻击成功、图像质量和可逆恢复；不依赖 LLM 或多卡训练；适合 Windows + RTX 4060 先做小规模复现。

**Cons 与改进方向：** README 过简、路径和尺寸硬编码较多、输入命名规则脆弱、集成模型显存占用较高、恢复验证入口不够显式。后续建议增加命令行参数、`requirements.txt`、单图冒烟脚本、恢复验证脚本、单模型/轻量集成对比实验和 Y/UV 消融实验。
