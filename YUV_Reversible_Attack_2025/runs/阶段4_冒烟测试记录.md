第一次运行已经完成模型权重下载并跑到攻击成功后的指标计算阶段，但在 `calMetrics.py` 报错：

```text
ValueError: win_size exceeds image extent...
If your images are multichannel (with color channels), set channel_axis to the axis number corresponding to the channels.
```

原因：

`calMetrics.py` 使用了旧版写法：

```python
skmetrics.structural_similarity(ori_np, rae_np, multichannel=True)
```

当前环境为 `scikit-image 0.26.0`，需要使用：

```python
channel_axis=-1
```
