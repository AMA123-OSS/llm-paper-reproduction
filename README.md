# YUV Reversible Adversarial Attack Reproduction

This repository is the cleaned reproduction package for the paper:

`Efficient and Transferable Reversible Adversarial Attacks Utilizing YUV Color Space`

## Project Layout

- `yuv-reversible-attack-reproduction/`: core code, evaluation scripts, and curated experiment results.
- `yuv-reversible-attack-reproduction/runs/`: staged reproduction records, CSV metrics, and final summaries.
- `项目分阶段执行计划与测试.md`: phased execution and testing checklist.
- `架构概览.md`: code architecture overview.

## What Is Preserved

- Original reproduction code for the YUV reversible adversarial attack.
- Small-scale local reproduction scripts and metrics.
- Y/UV ablation results.
- Transferability experiment results.
- Recovery-feasibility check results.
- Architecture overview and phased execution/testing plan.

## What Is Not Committed

The repository intentionally excludes heavy or nonessential artifacts:

- virtual environments
- downloaded model weights
- ImageNet input images
- generated RAE PNG files
- wheelhouse/cache folders
- temporary PDF extraction text
- old candidate-paper projects and notes

The committed CSV/MD summaries are the authoritative reproduction results for this cleaned package.
